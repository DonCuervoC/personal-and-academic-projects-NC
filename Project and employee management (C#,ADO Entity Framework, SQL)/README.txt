
 *********  WELCOME  ********

The objective of this application is to administer and control the development 
and management of projects, employees, contractors and materials for a company in the construction sector, 
this app is composed of a main interface in which the user has the possibility of entering the system as employee, 
supervisor or administrator, depending on the user's selection we will have a personalized session with different interfaces, 
permissions and utilities.

For this we use C#, ADO Entity Framework, SQL and LINQ as main development technologies.
